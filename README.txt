
Weather Information Retrieval Final Project - INST 126

What the program does
This Python script retrieves weather information for a specified city using the OpenWeatherMap API. It provides details such as temperature, wind speed, latitude, longitude, and a brief weather description.

How to use the program
Prerequisites
Before running the script, ensure you have:

Python installed on your machine.
An API key from OpenWeatherMap. You can obtain one by signing up on their website.

Setup
1.Clone the repository to your local machine.
2. Install the required Python packages using pip install requests
3. Replace YOUR_API_KEY in the script with your actual OpenWeatherMap API key.

Usage
1. Open a terminal in the project directory.
2. Run your script with projectname.py
3. Enter the city when prompted.
4. View the weather information displayed in the terminal.

Note
If you encounter any issues, make sure your API key is correctly configured, and you have an active internet connection.
